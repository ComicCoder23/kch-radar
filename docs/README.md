# KCH Radar

## What it is
A local radar for gigs, sober-friendly events, culture, retreats, and surrounding East Dunbartonshire opportunities.

## Why it matters
Helps people discover what is happening locally without information overload.

## Current Focus
- local event discovery
- sober filters
- retreat and wellbeing paths
- future transport/community layers

## Rule
All substantial KCH Radar work should live in this folder.
