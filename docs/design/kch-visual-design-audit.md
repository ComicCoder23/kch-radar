# KCH Visual Design Audit (Formal)

## 1. Executive Summary
The KCH project utilizes a "Live Radar" aesthetic that is visually impressive but technically focused. To become a community-led discovery platform, it requires a shift from "monitoring dashboard" to "local discovery".

## 2. Strengths
- High-quality, consistent dark-mode styling.
- Responsive layout with polished interactions.
- Strong typography hierarchy.

## 3. Weaknesses & Missing Identity
- Aesthetic is too "cyber/tech"; lacks the warmth of Kirkintilloch.
- Missing integration of local landscape and landmarks.
- Colour palette relies on high-tech green/blue; lacks natural local tones.
- Over-reliance on technical labels (e.g., "Signals", "Radar") which may alienate non-technical users.

## 4. Risks
- Accessibility: Dark-on-dark components are low-contrast in specific lighting.
- Readability: Technical typography (mono) should be reserved for specific labels.

## 5. Recommended Direction
Shift the aesthetic toward a warm, accessible, "Local Discovery OS". Introduce sandstone, canal-water, and natural tones to root the interface in the local environment while maintaining the current clean technical architecture.
