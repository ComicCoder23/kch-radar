# KCH Design System (Formal)

## 1. Colour Tokens
- **Canal Blue**: #0e4f77
- **Campsie Green**: #2d6a4f
- **Sandstone**: #d4a373
- **Page Bg**: #fdfdfc
- **Card Bg**: #ffffff
- **Primary Text**: #1a1a1a

## 2. Typography
- **Headings**: `Inter`, 700/800.
- **Body**: `Inter`, 400.
- **Labels**: `Inter`, 500.

## 3. Component Rules
- **Buttons**: Rounded corners, solid primary, clear borders for secondary.
- **Cards**: Soft shadows, 1px border.
- **Badges**: Pill-shaped, muted backgrounds.

## 4. Spacing
- 4px grid system. Section padding: 48px/24px.
