# KCH Community Chat / Kirky Rooms Feature Spec

## 1. Product Framing
KCH is a local discovery engine first. The Community Chat (Kirky Rooms) feature acts as a secondary fellowship layer designed to bridge the gap between "seeing an event" and "connecting with the community." It aims to foster local belonging in a moderated, safe environment.

## 2. Parent Inspiration (Yahoo-style)
- Publicly accessible thematic rooms.
- Pseudonymous handles.
- Fast, low-friction conversation.
- Emoji culture for emotional richness.

## 3. KCH-Safe Modernization
- **Moderation-First**: Auto-moderation, keyword flagging, and report mechanisms active from day zero.
- **Privacy-First**: No public DMs at MVP. Profiles are restricted to pseudonyms or interest-based badges.
- **Safety**: Robust community guidelines required for entry. No crisis handling; mandatory automated signposting to support resources.

## 4. Room Types
- **What's On Around Kirky**
- **Creative Kirkintilloch**
- **Local Help & Questions**
- **Campsie/Canal Walks**
- **New to Kirky Hub**

## 5. Gamification (Future)
- **Local Badges**: Canal badge, Campsie hiker, Community contributor.
- **Verified Organiser Tiers**: Badges for verified local group leaders.

## 6. Moderation & Safety
- **Report/Mute**: Standard tools for participants.
- **Guidelines**: Prominently displayed room rules.
- **Filtering**: Automated word-list blocking.

## 7. MVP Recommendation
**PHASED ROLLOUT:**
- **Phase 0**: Document Spec.
- **Phase 1**: Community noticeboard / comment-free discussion prompts.
- **Phase 2**: Pseudonymous rooms with moderation.
- **Phase 3**: Gamified badges and reputation.

## 8. Risks
- Moderation load, toxicity, safeguarding youth members, and distraction from the discovery engine.

## 9. Conclusion
The chat feature is a **future fellowship layer**. It should not be built until the discovery engine has real, frequently updated local data.
