# Components

This folder contains reusable UI building blocks for the KCH Radar frontend.

## Current components
- DiscoveryRadar
- FeaturedNow
- HiddenOpportunities
- LocalSignalsFeed
- CategoryGrid
- SubmitSignalForm

## Best practice rule
Each section should stay isolated so it can later be powered by:
- mock data
- Google Sheet JSON
- Apps Script feed
- API routes
