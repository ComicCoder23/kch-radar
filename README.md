# KCH Radar

KCH Radar is a discovery-first local website for Kirkintilloch and nearby towns.

## What it is
- local discovery radar
- events and opportunities frontend
- future Google Sheet + n8n powered site

## Current stack
- React
- Vite
- modular components
- mock queue data first

## Planned flow
Google Sheet -> Weekly Post Queue -> website -> Facebook Page

## Core homepage sections
- Discovery Radar
- Featured Now
- Hidden Opportunities
- Local Signals Feed
- Category Grid
- Submit Signal Form
